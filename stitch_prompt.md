# Google Stitch Prompt：柠檬字幕工作室 Android 版

## App 概述

一款 Android 离线字幕处理应用，中文化名称为"柠檬字幕工作室"（英文名：LemonSubtitle）。帮助用户将视频转字幕、翻译字幕、提取音频，全部离线运行。

目标用户：视频创作者、自媒体人、字幕制作人员。

---

## 底部导航（4 个 tab）

1. **串联工作室**（主页面，默认首页）— 转换工作台
2. **字幕编辑** — 视频播放 + 字幕精调
3. **模型管理** — 已导入的模型
4. **设置** — 应用配置

导航栏样式：Material 3 底部导航，选中项有填充色高亮，图标+文字标签。

---

## 页面 1：串联工作室（主页面）

### 功能描述
一站式文件转换入口。用户选择文件，选择操作类型，执行并查看进度。

### 布局（从上到下）

**顶部区域：操作选择器**
- 一行水平滚动的操作卡片，4 个：
  - 🎵 提取音频（图标：音乐符号）
  - 🎤 转字幕（图标：对话气泡）
  - 🌐 翻译字幕（图标：地球/语言）
  - ⚡ 全自动流水线（图标：闪电）

选中状态：卡片有描边高亮 + 背景浅色填充。

**文件选择区**
- 一个大按钮："+ 选择文件"（支持多选）
- 下方显示已选文件数："已选择 X 个文件"
- 文件支持类型：mp4, mkv, avi, mov, m4a, mp3, wav, srt, vtt

**任务队列列表**
- 每个任务一张卡片，横向布局：
  - 左侧：文件类型图标
  - 中间：文件名（单行省略）、文件大小（灰色小字）
  - 右侧：状态标签（等待中/处理中/完成/失败）
- 处理中的卡片显示进度条（水平）
- 空状态：显示插画 + "暂无任务，选择文件开始"

**底部配置栏（可折叠）**
- 展开后显示：
  - 目标语种选择（下拉：中文/英文/日文/韩文等）
  - 精度模式（下拉：极速/均衡/高精度）
  - 输出目录

### 交互流程
1. 用户点击"选择文件"→ 系统文件选择器弹出
2. 选完回到页面，生成任务卡片
3. 用户点操作卡片（如"转字幕"）→ 任务开始执行
4. 进度条更新，完成后状态变为"完成"

## 页面 2：字幕编辑

### 布局（竖屏）

**上半部分：视频播放器**
- 横屏 16:9 比例的视频画面
- 底部播放控制条：播放/暂停、进度条、全屏按钮、时间显示
- 全屏时视频占满屏幕，字幕列表浮在视频下方

**下半部分：字幕列表**
- 每行显示：
  - 时间码（灰色，如 00:12:34）
  - 原文文本（白色，可编辑）
- 点击某一行 → 视频跳转到对应时间点
- 点击文本 → 进入编辑模式，键盘弹出
- 失焦 → 自动保存

**底部操作栏**
- 导出按钮（单语 SRT / 双语 SRT / VTT）
- 双语预览开关

## 页面 3：模型管理

### 布局
- 页面标题："已导入模型"
- 空状态提示：引导用户下载 .gguf 文件
- 提供下载链接列表：
  - Whisper tiny（75MB）
  - Whisper base（142MB）
  - Whisper small（466MB）
  - 更多模型请访问 huggingface.co/ggerganov/whisper.cpp
- 已导入的模型用卡片展示：模型名 + 大小 + 删除按钮

## 页面 4：设置

### 布局
列表式设置项，每行一个：

- 默认目标语种（点击弹出选择器）
- 默认精度模式（极速/均衡/高精度）
- 默认输出目录
- 应用语言（跟随系统 / 中文 / English）
- 关于（版本号，开源信息）

## 设计风格

**色调：**
- 主色：#1A73E8（Google Blue）
- 背景：深色模式 #121212，浅色模式 #FFFFFF
- 卡片：深色 #1E1E1E，浅色 #F5F5F5
- 强调色：#34A853（成功/完成状态）
- 警告色：#EA4335（失败/错误状态）

**字体：**
- 中文：系统默认（Noto Sans SC fallback）
- 英文：Roboto
- 字幕列表用等宽字体显示时间码

**圆角：**
- 卡片和按钮：12dp
- 底部导航：默认 Material 3 规范

**间距：**
- 页面边缘：16dp
- 卡片之间：8dp
- 列表项内间距：12dp
