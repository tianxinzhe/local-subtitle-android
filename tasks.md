# 《柠檬字幕工作室》Android 版实现计划

## Task 1: Android 项目初始化
- **优先级**: P0
- **依赖**: 无
- **描述**:
  - 创建 Android Studio 项目（Kotlin + Jetpack Compose + Material 3）
  - 配置 NDK + CMake 编译环境
  - 集成 whisper.cpp 源码（cmake 编译为 ARM64-v8a）
  - 集成 ffmpeg-android（预编译 .so 库）
- **验证**: 项目构建成功，native C++ 库正确加载

## Task 2: 算力检测模块
- **优先级**: P0
- **依赖**: Task 1
- **描述**:
  - 检测 ARM GPU 型号（Mali/Adreno/PowerVR）
  - 检测 CPU 核心数，配置线程池
  - 任务开始前检测一次，运行中不做 GPU 热切换
- **验证**: GPU/CPU 检测正确，降级不崩溃

## Task 3: 主界面与导航
- **优先级**: P0
- **依赖**: Task 1
- **描述**:
  - 底部导航栏 4 个 tab：串联工作室、字幕编辑、模型管理、设置
  - Tab 切换、页面容器
  - 所有文案引用 strings.xml，无硬编码
  - Jetpack Compose + Material 3 原生 UI
- **验证**: 4 个 tab 切换正常

## Task 4: 串联工作室（主页面）
- **优先级**: P0
- **依赖**: Task 3
- **描述**:
  - 系统文件选择器（音视频/字幕文件，支持多选）
  - 操作选择器：提取音频 / 转字幕 / 翻译字幕 / 全自动流水线
  - 任务列表：文件名、大小、状态、进度条、剩余时间
  - 全局配置：语种选择、模型选择、输出目录
  - 用户可自主选择 Whisper 模型（tiny/base/small）
- **验证**: 文件选择→选操作→执行→出结果，全流程跑通

## Task 5: 音频提取模块
- **优先级**: P0
- **依赖**: Task 1
- **描述**:
  - ffmpeg-android（C++ 库，通过 JNI 调用）
  - 参数：16kHz 单声道 WAV
  - 导出格式：MP3 / WAV / AAC / OGG
  - 比特率：128k-320k
  - 状态回调到 Kotlin UI
- **验证**: 各格式导出正确，比特率按设置生效

## Task 6: Whisper.cpp ASR 集成
- **优先级**: P0
- **依赖**: Task 1, Task 5
- **描述**:
  - whisper.cpp 源码纳入项目，CMake 编译
  - Kotlin 通过 JNI 调用 whisper.cpp
  - **用户通过提供的模型下载链接自行下载 .gguf 文件**
  - 支持全部模型尺寸，不做限制（tiny/base/small/medium/large 均可）
  - 用户导入模型文件后应用自动识别可用
  - 自动检测音频语言，用户可手动修正
  - 30 秒幻听截断保护
  - 临时文件自动清理
- **验证**: 用户下载导入的 .gguf 模型能正常加载并转写

## Task 7: 字幕解析器
- **优先级**: P0
- **依赖**: Task 1
- **描述**:
  - 解析 SRT / VTT 格式
  - 兼容断档行号、样式代码、人名标记
- **验证**: 标准和不规范字幕都能正确解析

## Task 8: 翻译模块
- **优先级**: P0
- **依赖**: Task 7
- **描述**:
  - **首发**：使用 Google ML Kit 翻译 API（离线翻译，支持 60+ 语言）
  - 一行 Gradle 依赖引入，无交叉编译成本
  - ML Kit 自动管理翻译包下载与缓存
  - 支持中英互译及小语种翻译
  - **后期规划**：支持用户下载自定义翻译模型并加载使用
- **验证**: ML Kit 各语种翻译结果正确

## Task 9: 字幕编辑页面
- **优先级**: P0
- **依赖**: Task 3, Task 6, Task 7
- **描述**:
  - ExoPlayer 视频播放器
  - 字幕列表（时间轴 + 文本）
  - 点击字幕跳转到对应时间点
  - 修改字幕文本，失焦即保存
  - 双语合成预览
  - 导出：单语 SRT / 双语 SRT / VTT
- **验证**: 播放→编辑→导出全流程跑通

## Task 10: 模型管理页面
- **优先级**: P1
- **依赖**: Task 3
- **描述**:
  - 模型列表：显示已导入的 Whisper 模型
  - 每个模型显示名称、大小、语言
  - 提供下载引导（展示官方下载地址，应用内不提供下载）
  - 用户从外部下载 .gguf 文件后，通过应用导入并注册
  - 已导入模型可删除
  - 保存到 filesDir/models/

## Task 11: 批量队列
- **优先级**: P0
- **依赖**: Task 4, Task 5, Task 6, Task 8
- **描述**:
  - WorkManager 管理后台任务
  - ForegroundService + 通知
  - 全自动流水线：提取→转写→翻译→导出→清理→下一个
  - 文件命名：[原名].bilingual.srt
  - 用户自行决定是否使用
- **验证**: 多个任务串行跑完，通知正确

## Task 12: 异常处理
- **优先级**: P0
- **依赖**: Task 6, Task 11
- **描述**:
  - 无声视频 60 秒超时跳过
  - 磁盘满时暂停队列 + 弹窗
  - 字幕时间轴重叠冲突处理
- **验证**: 三种场景处理正确

## Task 13: 多语言支持
- **优先级**: P1
- **依赖**: Task 3
- **描述**:
  - 所有 UI 文案使用 Android strings.xml 定义，代码中无硬编码字符串
  - 创建中文 values-zh 和英文 values-en 两个基础语言包
  - 英文作为 fallback 默认语言（values/strings.xml）
  - 设置页支持手动切换语言
  - 新增语言只需在 res/values-{lang}/ 下添加 strings.xml，无需改 Kotlin 代码
- **验证**:
  - 切换语言后所有界面即时更新
  - 新增一个假语言包验证无需改代码
  - 所有可达界面文案全部走 strings.xml，无遗漏

## Task 14: 设置页面
- **优先级**: P2
- **依赖**: Task 3
- **描述**:
  - 默认语种（翻译目标语言）
  - 默认模型
  - 默认输出目录
  - 语言切换（系统跟随 / 手动选择）
  - 关于页面
  - DataStore 持久化
- **验证**: 设置修改后重启仍保留

## Task 15: APK 打包
- **优先级**: P0
- **依赖**: 所有核心功能
- **描述**:
  - ARM64-v8a 单 ABI 打包
  - ProGuard 混淆
  - ≤50MB
- **验证**: APK 构建成功，安装运行正常
